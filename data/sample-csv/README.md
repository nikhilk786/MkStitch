# Kurti Sample CSV Upload

1. The CSV files are configured for `https://ik.imagekit.io/nicks`.
2. Upload category images and product images to ImageKit with matching names.
3. Upload `kurti-categories.csv` from the admin category bulk upload page.
4. Upload `kurti-products.csv` from the admin product bulk upload page.
5. Verify products in admin and on the frontend product listing/detail pages.
